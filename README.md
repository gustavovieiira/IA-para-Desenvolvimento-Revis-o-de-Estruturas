# IA para Desenvolvimento + Revisao de Estruturas de Dados I

Este repositorio foi organizado para a atividade de revisao de Estruturas de Dados I com uso obrigatorio de IA no desenvolvimento.

## Estrutura

- `relatorio.md`: relatorio principal para entrega
- `arrays/`: 5 problemas de arrays
- `linked_lists/`: 4 problemas de listas encadeadas
- `stacks/`: 3 problemas de pilhas

## Linguagem

Python 3

## Problemas escolhidos

### Arrays

1. Two Sum
2. Best Time to Buy and Sell Stock
3. Product of Array Except Self
4. 3Sum
5. Merge Intervals

### Listas Encadeadas

1. Reverse Linked List
2. Linked List Cycle
3. Add Two Numbers
4. Remove Nth Node From End of List

### Pilhas

1. Valid Parentheses
2. Min Stack
3. Daily Temperatures

## Como executar

Cada arquivo pode ser executado individualmente:

```bash
python arrays/two_sum.py
```

ou

```bash
python linked_lists/reverse_linked_list.py
```

Os arquivos incluem exemplos simples no bloco `if __name__ == "__main__":`.
